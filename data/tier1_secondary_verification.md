## §4 公式 KO ページ翻訳品質判定

### 4.1 NEW ユーアイ GC (https://m-uigc.com/kr/index.html)

| 観点 | 評価 | 根拠 |
|---|---|---|
| 助詞使い分け | 自然 | 「청바지와 샌들 차림으로 입장 및 플레이는 삼가」等、文脈に合っている |
| ゴルフ専門用語 | 良好 | 「티잉그라운드」「벙커」「미들홀」「투온/원온」が適切 |
| 地名処理 | 一貫 | 「玄界灘(겐카이나다)」「津屋崎(츠야자키)」の音写方針が一貫 |
| 機械翻訳明示 | なし | 「자동번역」「Google 번역」等の表示なし |
| 改行・段落 | 自然 | 文中での不自然な改行なし |

**中村判定**: **Tier 1.5** (公式 KO 直対応・独自翻訳の可能性高い・ただしネイティブ確認なしに Tier 1 確定不可)

### 4.2 JR 内野 CC (https://www.jruchino.cc/apps/kr)

| 観点 | 評価 |
|---|---|
| 「컴룸」「숙녀 욕실」存在確認 | ✓ 確認 |
| 翻訳ツール埋込み (Google Translate API 等) | なし |
| 「자동번역」明示 | なし |
| 翻訳の自然さ | 不自然 (「컴룸」=コンペルームの直音化短縮・「숙녀」は古めかしい) |

**中村判定**: **Tier 3** (機械翻訳または低品質人的翻訳・「자동번역」明示推奨)
