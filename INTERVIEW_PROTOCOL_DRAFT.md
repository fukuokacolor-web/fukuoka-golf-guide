## 1. 打診メールテンプレ (3 バージョン)

### 共通の編集方針 (山本の編集判断)

- **媒体名 + 月間 PV を冒頭で明示**: 信用力獲得 (現状 500-1,500 PV/月だが、誇張せず実数を提示する原則)
- **取材意図を 3 行以内で要約**: 多忙な広報・支配人の可読性を優先
- **「掲載確定」を約束しない**: 編集独立性を維持。「掲載候補として企画進行中」が誠実
- **校正フローを明記**: 取材後に発言内容の確認を約束 → 信頼形成
- **アフィリ言及は控えめに**: 編集と広告を分離する原則。CTA は記事下端で別セクション
- **広告色を排除**: 「貴社の宣伝記事を作ります」と読まれた瞬間に断られる。「読者にコース選びの一次情報を届けたい」を主軸にする
- **添付**: 媒体プロフィール 1 枚 PDF (要別途作成・本草案ではフォーマット案のみ提示)

## 2. 取材ガイド雛形 (15-20 分インタビュー)

### 共通フォーマット

| 項目 | 内容 |
|---|---|
| **取材時間** | 15-20 分 (オンライン/訪問) + 文字起こし用録音許諾確認 5 分 |
| **準備物 (取材者側)** | IC レコーダー or スマホ録音アプリ × 2 (バックアップ含む) / メモ用紙 / 名刺 / 媒体プロフィール PDF (印刷) / 質問リスト印刷版 |
| **服装** | ビジネスカジュアル (ジャケット推奨・コース内取材時は襟付シャツ + チノパン・スパイク不要) |
| **同行者** | 編集者 1 名 + 撮影者 1 名 (撮影者は外注 or 編集者兼任) |
| **NG ワード/態度** | 「他コースとの優劣比較を要求」「アフィリ単価交渉」「報道的批判取材」を絶対しない |

### 質問リスト (10 個・90% は 5-7 個に絞って実施)

**Phase 1: 関係構築 (2 分)**
1. 本日はお忙しい中、お時間をいただきありがとうございます。まず、貴クラブ/コースとの関わりはどのくらいになりますか?

**Phase 2: コース・運営の核 (8-10 分)**
2. このコースの設計思想・最も特徴的なホールを 1 つ教えていただけますか?
3. 福岡県内の他コースと比較したときの、貴コースのユニークな価値は何でしょうか?
4. 過去 5 年間で実施された改修・サービス向上で、最も読者に伝えたいことは?
5. (大手チェーン向け) 全国ネットワークの中での福岡コースの位置付けは?
6. (会員制向け) ビジター来場者を歓迎する条件と、当日の流れを教えていただけますか?

**Phase 3: 来場者・読者向け (3-5 分)**
7. 初級者・中級者の方が来場されたとき、特に注意してほしいホールやマネジメントは?
8. 最近のインバウンド (海外) からの来場の現状はいかがですか? 言語サポートで困った経験は?
9. 予約 (じゃらん/楽天/公式) を検討中の読者に、どのチャネルが特におすすめでしょうか?

**Phase 4: 締め (2 分)**
10. 福岡で初めてゴルフ場を選ぶ読者へ、メッセージをいただけますか?

### 撮影方針

| 撮影対象 | 数量 | 用途 |
|---|---|---|
| 取材者ポートレート (顔出し OK の場合) | 2-3 枚 | 記事冒頭・JSON-LD Person Schema 用 |
| クラブハウス外観 | 1-2 枚 | 記事 hero 候補 |
| 取材者の動きが入ったショット (案内・解説中) | 2-3 枚 | 記事中段配置 |
| 特徴的なホール (本人推薦の 1 ホール) | 1-2 枚 | 「設計思想」セクション挿絵 |

**撮影権利**: 撮影写真は媒体側が著作権を保持。取材先には「貴社公式サイト・SNS で本記事へのリンク付き引用利用は自由」と明記する → 被リンク獲得導線を仕込む

**取材者顔写真の同意取得**: 「掲載に同意」の口頭確認 + 校正時にメール再確認 (ダブルチェック)

## 4. リスク・遵守事項

### 4.1 取材後の校正フロー (公開前必須)

```
取材日 → 録音文字起こし (3-5 営業日) → 編集者ドラフト (3-5 営業日)
→ 編集部内ファクトチェック (鈴木一郎 + 中村麻衣・1-2 営業日)
→ 取材者校正 (5-7 営業日・先方ご都合)
→ 修正反映 + JSON-LD 検証 (1 営業日)
→ 公開
```

**取材者校正で守る原則**:
- ✅ 発言の趣旨修正・事実誤認訂正は **全面的に受け入れる**
- ✅ プロフィールの追加・削除依頼は受け入れる
- ⚠️ 競合他社批判・断定表現の削除依頼は受け入れる
- ❌ **「ポジティブ表現の追加要請」(媒体側の編集判断を超える広告化要請) は丁寧にお断りする** → 編集独立性の根幹

### 4.2 公開前確認チェックリスト

- [ ] 取材者の校正済原稿との差分ゼロ確認
- [ ] JSON-LD InterviewSchema が Schema Markup Validator でエラーなし
- [ ] 写真の著作権表記漏れなし
- [ ] アフィリリンクと編集本文の視覚的分離 (border / セクション分け / 「PR」表記)
- [ ] 「本記事は編集独立性に基づく取材記事です。アフィリエイトリンクは記事末尾の予約セクションのみに掲載しております」のフッター文言挿入
- [ ] 関連記事 3 件の整合性確認 (同エリア・同価格帯)
- [ ] hreflang・canonical・OGP 設定確認

### 4.3 編集独立性の維持

- ✅ 取材謝礼: **金銭授受なし**。書籍 (媒体ガイドブック) 進呈程度に留める。お土産 (お菓子等 ¥3,000 以内) は許容
- ✅ 接待: **昼食代は媒体側が支払う** (取材先に支払わせない)
- ✅ 取材内容と広告掲載の分離: **「取材記事」と「広告記事 (PR)」を URL 構造で分離**
  - 取材記事: `interview-{slug}.html`
  - 仮に有料広告化する場合: `pr-{slug}.html` + 冒頭で「PR」明示
  - **当面は PR 記事は受けない方針** (山本判断)
- ✅ 取材先からの「自社のここを書いてほしい」要望は参考に留め、**最終的な編集判断は媒体側**
- ✅ 公開後 6 ヶ月以内の自社アフィリ単価変動と記事内容の連動を **禁止** (編集と営業の分離)

### 4.4 有料広告との分離 (最重要)

| 区分 | 取材記事 | 通常コースページ | 広告記事 (将来想定) |
|---|---|---|---|
| URL | `interview-{slug}.html` | `course-{slug}.html` | `pr-{slug}.html` |
| アフィリリンク | 記事末尾セクションのみ | 既存通り | 記事末尾セクションのみ |
| 「PR」表記 | 不要 (編集記事) | 不要 | **記事冒頭に必須** |
| 取材謝礼 | 金銭なし | — | (将来規定要) |
| 校正方針 | 趣旨修正のみ受入 | — | 取材先全権 |

### 4.5 リスクシナリオと対応

| リスク | 発生確率 | 対応 |
|---|---|---|
| 取材後に取材先が掲載拒否 | 中 | **取材時に「掲載確約はしない・媒体判断」を口頭確認**。発生時はその記事を放棄し補欠コースへ振替 |
| 校正で大幅書換要求 | 中 | 山本編集長の判断で線引き。**広告化要求は丁寧拒否**・事実訂正は受入 |
| ファクトチェックで重大誤りが発覚 | 低-中 | 公開直前に鈴木・中村が再確認。**疑わしきは公開延期** |
| 公開後に取材先からクレーム | 低 | 即日校正履歴と取材録音を確認。事実誤認なら即訂正・取材先過失なら丁寧に経緯説明 |
| 競合他社からの「同等の取材機会要請」 | 中 | **連載 5 件の選定基準を明文化** (`INTERVIEW_CANDIDATES.md` のスコアリング) → 透明性で対応 |
| 取材謝礼を要求された | 低 | **金銭授受は方針で全件拒否**。代替として「貴クラブ公式サイトへの被リンク + 媒体内特集ページへの紐付け」を提案 |
